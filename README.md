# Internship Project Summary

This repository summarizes my contributions during my internship on a **Task Management Application**, including both frontend and backend development, using standard development tools.

---

## Project Overview

The application allows users to create, organize, and track tasks. Features include reminders, categories, and file attachments.

I have worked on both the **Flutter frontend** and **backend APIs**, implementing features and making improvements.

---

## Frontend (Flutter)

**Tech used:** 
- Flutter (Dart)
- REST APIs  

### Contributions
- Developed screens for task creation, editing, deletion, and completion  
- Implemented reminders with date and time selection  
- Organized tasks by categories  
- Added support for file attachments  
- Integrated backend APIs for real-time updates  

### Improvements
- Fixed UI issues such as layout overflows  
- Improved image rendering  
- Refactored task home screen for better structure  

---

## Backend

**Tech used:** 
- JavaScript (Node.js)
- Python
- PostgreSQL
- Sequelize (ORM)
- REST APIs
- Docker

### Contributions
- Built APIs for task creation, update, deletion, and listing  
- Added task filtering options  
- Updated database structure for new features  
- Implemented file upload support  
- Developed task grouping logic
- Used Sequelize ORM for defining models and interacting with PostgreSQL database

### Improvements
- Simplified API structure for maintainability  
- Ensured smooth integration with frontend  

---

## Tools & Workflow
- **Git** for version control and collaboration  
- **Docker** for containerizing backend and database services to ensure consistent development environments

## Summary
